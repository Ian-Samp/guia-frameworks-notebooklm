# Guia para Frameworks | NotebookLM
## 🎯 Objetivo
### Contexto
Este projeto é o desafio **Treinando uma IA de Aprendizagem** do **Bootcamp Bradesco — GenAI, Dados & Cyber**, que tem como objetivo demonstrar de forma prática  minha capacidade de aliar pensamento crítico, organização, conhecimento de Modelos de Linguagem (LMs) e aplicação eficiente de IAs generativas, construindo um  Caderno Temático que reúna fontes relevantes sobre um tema para estudos na plataforma NotebookLM. Esta plataforma permite centralizar fontes conviáveis sobre um tema, que funciona como um banco de dados personalizado para a LM, garantindo uma confiabilidade maior para as fontes que a IA irá utilizar para gerar  respostas e mitigando ocorrências de "alucinações de IA" (respostas aparentemente confiáveis, mas que são totalmente falsas).
### Frameworks de Cibersegurança
O tema escolhido para o Caderno Temático foi **"Frameworks de Cibersegurança"** (conjuntos de estruturas conceituais, metodologias e padrões para organizar defesas, analisar ataques e gerenciar riscos). Meu objetivo com o tema é relacionar meus estudos atuais de Segurança da Informação com o desafio do Bootcamp.
