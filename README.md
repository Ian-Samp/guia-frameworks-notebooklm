# Guia para Frameworks | NotebookLM
## 🎯 Objetivo
### Contexto
Este projeto faz parte do desafio **Treinando uma IA de Aprendizagem** integrante do **Bootcamp Bradesco — GenAI, Dados & Cyber**. O objetivo é demonstrar de forma prática a capacidade de aliar pensamento crítico, curadoria de conteúdo e aplicação de IA generativa.

Para isso, construí um Caderno Temático na plataforma **NotebookLM**. A ferramenta funciona centralizando fontes confiáveis como um banco de dados personalizado, o que garante maior acurácia nas respostas da IA e mitiga significativamente o risco de "alucinações".
### Frameworks de Cibersegurança
O tema escolhido para o Caderno Temático foi **"Frameworks de Cibersegurança"** (conjuntos de estruturas conceituais, metodologias e padrões para organizar defesas, analisar ataques e gerenciar riscos). Meu objetivo com o tema é relacionar meus estudos atuais de Segurança da Informação com o desafio do Bootcamp.
### Metas
Ao finalizar este projeto espero conseguir:
- [ ] Mapear e categorizar pelo menos 3 frameworks de cibersegurança essenciais;
- [ ] Utilizar uma IA contextualizada via RAG (Retrieval-Augmented Generation) para atuar como um tutor especializado nos frameworks estudados;
- [ ] Documentar o processo de troubleshooting (cicatrizes) enfrentado ao refinar os prompts para a IA;
- [ ] Desenvolver uma estrutura replicável de estudos que possa ser aplicada a outras verticais de Segurança da Informação;
- [ ] Criar um dicionário de termos técnicos (Glossário) com o suporte da IA para fixação de conceitos complexos.
